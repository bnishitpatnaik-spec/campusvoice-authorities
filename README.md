# CampusVoice Admin

## Official Control Center

Management dashboard for campus authorities to manage and resolve student complaints.

## Features

- View all complaints in real time
- Change complaint status: Pending → In Progress → Resolved
- Upload resolution proof photos
- Faculty endorsed complaints prioritized
- Analytics and statistics
- Manage students and faculty
- Send push notifications
- Auto reject complaints without location
- Satisfaction rating from students

## Tech Stack

- Frontend: React, TypeScript, TailwindCSS
- Backend: Node.js, Express, TypeScript
- Database: Firebase Firestore
- Images: Cloudinary

## How to Run

Terminal 1 - Backend:
```
cd backend
npm run dev
```

Terminal 2 - Frontend:
```
npm run dev
```

Open: http://localhost:8081
